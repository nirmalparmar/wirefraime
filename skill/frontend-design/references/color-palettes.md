## How to Apply

When choosing a palette for an app:
- **Finance/Banking** → Slate Titanium, Onyx Sapphire, Champagne Gold
- **Fitness/Health** → Obsidian Emerald, Carbon Coral, Charcoal Teal
- **E-commerce/Luxury** → Champagne Gold, Mink Rosegold, Midnight Luxe
- **Productivity/SaaS** → Slate Titanium, Arctic Frost, Deep Ocean
- **Creative/Portfolio** → Velvet Plum, Royal Indigo, Graphite Peach
- **Medical/Wellness** → Glacier Mint, Ivory Sage, Arctic Frost
- **Social/Consumer** → Pearl Rose, Carbon Coral, Deep Ocean
- **Food/Restaurant** → Espresso Cream, Noir Amber, Graphite Peach

Map palette roles to design system colors:
- Base → background
- Surface → surface (cards, panels)
- Primary/Accent → primary
- Deep variant → use for hover states, active nav, emphasis
- Light/Pale → use for badges, tinted backgrounds, subtle highlights
- Derive textMuted from midpoint between text and surface
- Derive border from surface with slight opacity shift
